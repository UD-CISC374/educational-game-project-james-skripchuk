Big O Notation

Definition of Big O:

\(f(n)\) is said to be \(O(g(n))\) if and only if

there exists positive constants \(n_0\) and \(c\) such that

\(f(n) \leq cg(n)\) for all \(n \geq n_0\)

Okay, that's a lot of math notation at once. Here's an example:

\(f(n) = 2n^2+4n\)

\(c = 4\)

\(g(n) = n^2\)

\(n_0 = 2\)

\(2n^2+4n\) is \(O(n^2)\), because \(2n^2+4n \leq 4n^2\) for all \(n \geq 2\)

As you can see, the functions intersect at \((2,16)\).

It is at this point, \(cg(n)\) overtakes \(f(n)\), fulfilling the conditions for \(f(x)\) to be \(O(g(x))\)

Remember, \(n_0\) and \(c\) can be anything, as long as the statement \(f(n) \leq cg(n)\) for all \(n \geq n_0\) holds.

In the above example, the Big O condition also holds true for \(c=4\) and \(n_0=3\).

But, it is also nice to have a procedure to follow when proving something; so this is how I arrived at the values \(c=4\) and \(n_0=2\) for the above example

Calculation Example

Since we can prove Big O if it holds with any \(c\) and \(n_0\), we can pick any \(n_0\) we want, and solve for it's respective \(c\)

We choose \(n_0=2\)

Now, we can substitute our variables into the Big O condition:

\(f(n) \leq cg(n)\)

\(2n^2+4n \leq cn^2\)

\(2(2)^2+4(2) \leq c(2)^2\)

\(8+8 \leq 4c\)

\(16 \leq 4c\)

\(4 \leq c\)

I chose \(c=4\) since it is the smallest possible \(c\) that works, but notice that you can choose any \(c\) as long as it is greater than 4.

\(2n^2+4n\) is \(O(n^2)\), because \(2n^2+4n \leq 4n^2\) for all \(n \geq 2\)
